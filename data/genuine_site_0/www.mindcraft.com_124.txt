         <h2 align="left"><strong>The LDAP Server Benchmarking Tool</strong></h2>
         
    <h2>
    <img src="../webstone/new.gif" width="38" height="39">DirectoryMark 1.3</h2>
         
      <p>DirectoryMark 1.3 has support for  Active Directory on Windows 2000 and 
      Windows Server 2003. It also has support for Active Directory in 
      Application Mode as well as 
        a new set of <a href="dm13runrules.html">run rules</a> that eliminates 
      the need for directory size classes. The <a href="history.txt">change 
      history</a> for DirectoryMark 1.3 is available for your review. We have released versions for Windows and Solaris. Download 
        the one you want now! The <a
         href="dirmark-exec-license.html">executables are free</a>. If you are 
        interested in the source code, please <a
         href="mailto:sales@mindcraft.com">contact us</a>.</p>
         
      <p>The DirectoryMark 1.3 benchmark is designed to measure the performance 
        of server products that use Lightweight Directory Access Protocol (LDAP), 
        version 3.</p>
         <p>You can customize the DirectoryMark testing scenarios to meet your
         needs. You can use your own data in LDIF format to generate test
         scripts.</p>
         <p>Mindcraft is maintaining DirectoryMark and is providing this Web
         site as a service for our clients and for the industry. We would
         appreciate any <a href="mailto:directorymark@mindcraft.com">feedback</a>
         you have.</p>
