What it is?

Sensors Server is a server process that detects and reads your hardware's health monitoring chipset and streams the data across a network socket for client applications to connect to and process. Included with this package are several Java applets for viewing your sensors live via your favorite Web browser, a watchdog utility that will execute a user process if and when alarm conditions occur, and a command line client utility useful for scripting and/or testing.
