Canabalt


Genre: Flash Platform Games
Developer: Adam Atomic and Danny B.


Game Overview:
Canabalt is a simple one-button game which resembles a sort of free-running or 'parkour' style platform game. Stylized graphics run fluidly across the screen as you attempt to jump from one rooftop to another to escape the crumbling city.

Small additions to the game are a nice touch which include animated robots in the background destroying the city, as well as flocks of birds flying up as you run past at blazing speeds. The longer you run without hitting obstacles, the faster the game gets until the inevitable happens.

Controls / How To Play:
X or C = Jump
