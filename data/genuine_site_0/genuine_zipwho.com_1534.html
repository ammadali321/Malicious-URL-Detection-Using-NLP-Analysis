About ZipWho.com

Welcome to ZipWho.com!

This ZIP code demographics site was built because existing census statistics tools failed to satisfy the needs of power users. Many Web sites allow users to query the median household income of a ZIP code, but that dollar figure alone does not indicate how one neighborhood's population compares to others around the country. ZipWho.com fills this information gap by displaying a national percentile rank (0 - 99) next to every data element.

In addition to basic ZIP code lookup functionality, this site allows users to search by demographics. Marketing professionals can appreciate this powerful segmentation feature that is not offered by any other Web site. So go ahead and play with the search fields on the left. All of the searches are self-contained in the URL at the top of your browser. That means you can easily bookmark your searches and email them to your friends.
